Title : Pharmacy Database Project
Author : Matt Reid
Date : March 31st, 2026

Program design - This program is designed to allow a user to add prescriptions, drugs, and patients to a pharmacy database.

Program requirements:

- Java 17+ installed
- PostgreSQL installed and running
- PostgreSQL JDBC driver added to your project(via Maven or Manually)

Usage Instructions:

1. Compile the code in the terminal:
   - javac SRC/Main.java
2. Run the Program:
   - javac SRC.Main

Using the menu:

- You will have a menu appear in the terminal that has the following:
  1. Save Drug to File
  2. Read Drugs from File
  3. Save Patient to Database
  4. Read Patient from Database
  5. Exit

- In menu item 1 - You have to enter a Drug ID (must be numerical values only), Drug Name, Drug Cost (Cannot be a negative value), Drug Dosage. The Drug will then be saved to the drug.txt file for viewing later if needed.

- In menu item 2 - You simply select the number 2 from the menu and it will display all of the drugs that have been entered.

- In menu item 3 - You have to add the Patients information in the system. This will be saved in the database. You must enter a Patient ID (must be a numerical value), First Name, Last Name, DOB of the patient.

- In menu item 4 - You simply select the number 4 and it will display all of the patients in the database.

- Menu item 0 - Exits the program.
